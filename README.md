## Xcode Preferences

### Usage

```sh
# Xcode Preferences

make apply_preferences

# Installable resources

make install_tempaltes
make install_snippets
make install_themes

# Cleaning (removes everything!)

make clean_templates
make clean_snippets
make clean_themes

# Conveniences

make clean_install_templates
make clean_install_snippets
make clean_install_themes
```
